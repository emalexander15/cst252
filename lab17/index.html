OOP: Hangman

1. Understand the Problem

  • Think of a word or phrase
  • Draw the lines
  • Have a friend(s) guess 1 letter at a time
  • Every incorrect score add a body part
  • Guess until correct or stick figure is completed

    • 2. Pseudocode the High Level Approach

    Set up the game initially
	Think of a word or phrase
	Draw the number of lines corresponding to the letters
Start the game loop
	Ask friend(s) to give letters
	Write correct letters on provided spaces
If the letter is wrong
	Begin drawing the stick figure
	Draw a body part for every incorrect letter
If the game should end
	Display the proper victory
Else
	Keep looping with new letters until phrase or
	figure completed

    3. Assign Actions to Objects

    Set up the game initially [Hangman]
	Think of a word or phrase [Player]
	Draw the number of lines corresponding to the letters [Board]
Start the game loop [Hangman]
	Ask friend(s) to give letters [Player]
	Write correct letters on provided spaces [Board]
If the letter is wrong [Board]
	Begin drawing the stick figure [Player]
	Draw a body part for every incorrect letter [Player]
If the game should end [Hangman]
	Display the proper victory [Board]
Else
	Keep looping with new letters until phrase or
	figure completed [Player]
